JQuery DataTable Column Filter - Using Regular Expressions

Preamble

DataTables Column Filter plugin can be used to initialize regular expression searches. In the regular expression search you can enter regular expression with various wildcards such as:

  • Start of the line ^
  • Any character .
  • Any character in the range between characters N and P [M-P]
  • End of the line $
    Also you can use operators:
  • Character/strings that repeats 0 or more times - operator *
  • Character/strings that repeats 1 or more times - operator +
  • Character that may but does not need to appear in the sequence - operator ?
    Here are some examples you might try
  • Any sequence of zero or more characters - .*
  • Browser versions that ends with 0, 1, or 2 - [0-2]$
  • Platforms starting with OSX - ^OSX
  • Win 98 optionally followed byt the SE - Win 98(SE)?
  • Platworms containing word "Win" followed by any sequence of characters and then OSX (e.g. "Win 95+ / OSX.1+")- Win.*OSX
  • Browsers that starts with "ca" and ends with "0" (e.g. "Camino 1.0") - ^ca.*0$

Initialization code

To setup regular expression searches just pass bRegex parameter to the initialization parameters in the text searches:

			$(document).ready( function () {
			    $('#example').dataTable().columnFilter({ aoColumns: [
                                                            { type: "text", bRegex:true },
                                                            { type: "select",	bRegex:true, 
                                                            values: [
                                                            { value: '^(Internet|IE)', label: 'Internet Explorer(any)'},
                                                            { value: '^Mozilla', label: 'Mozilla(any)'},
                                                            { value: '^Firefox', label: 'FireFox(any)'}
                                                            ]
                                                            },
                                                            { type: "text", bRegex:true },
                                                            { type: "text", bRegex:true },
                                                            { type: "text", bRegex: true }
                                                        ]});
				
			} );
