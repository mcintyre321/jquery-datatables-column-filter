JQuery DataTable Column Filter - Server Side Example with delayed filter

Preamble

In the server-side mode it might be a performance issue if plugin sends request each time user types a new charater. To prevent this you can set iFilterLength parameter in each column. Filtering will not be performed until user types a leasts iFilterLength characters in the filter text box.

Many thanks to Mike Robellard for this code
